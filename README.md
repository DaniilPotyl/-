## Описание проекта

Telegram-бот с мини-играми, написанный на Python с использованием библиотеки **aiogram 3**.

Бот позволяет пользователям играть в крестики-нолики против бота и угадывать загаданное число. За победы начисляются очки, которые можно тратить в магазине на полезные предметы.

Все данные хранятся в локальной базе данных SQLite (`db.db`).

### Возможности
- Игра в крестики-нолики против бота
- Игра "Угадай число" (от 1 до 100)
- Система очков за победы и ничьи
- Магазин предметов за очки
- Инвентарь с активными предметами

## Структура проекта

```
Шаблон бота/
├── main.py           # Основной файл бота (хэндлеры, логика игр, магазин)

```

### Таблицы базы данных

| Таблица | Назначение                                          |
|---------|-----------------------------------------------------|
| `users` | Данные пользователей (очки, статус, инвентарь, VIP) |

### Система очков

| Событие                      | Очки |
|------------------------------|------|
| Победа в крестики-нолики     | +50  |
| Ничья в крестики-нолики      | +15  |
| Поражение в крестики-нолики  | +0   |
| Угадал число                 | +20  |

### Предметы магазина

| Предмет             | Цена | Описание                                          |
|---------------------|------|---------------------------------------------------|
| ⚡ Двойные очки     | 200  | Следующая победа приносит x2 очков (разовый)      |
| 💡 Подсказка        | 150  | Диапазон ±10 в "Угадай число" (разовый)           |
| 🛡 Щит              | 300  | При поражении в крестики даёт +25 очков (разовый) |
| 👑 VIP статус       | 1500 | Постоянный множитель x1.5 ко всем очкам           |

## Инструкция по запуску на Windows

### 1. Клонирование / копирование проекта

Если проект уже есть на рабочем столе, этот шаг можно пропустить.

Если используете GitHub:
```bash
git clone https://github.com/ваш_репозиторий
cd "Шаблон бота"
```

### 2. Создание и активация виртуального окружения

```powershell
# Создать виртуальное окружение (папка venv появится в корне проекта)
python -m venv venv

# Активировать окружение
.\venv\Scripts\Activate.ps1
```

При успешной активации в начале строки PowerShell появится префикс `(venv)`.

> **Если PowerShell выдаёт ошибку выполнения скриптов**, выполните один раз:
> ```powershell
> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
> ```

### 3. Установка зависимостей

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### 4. Настройка токена

Откройте файл `config.py` и вставьте токен:

```python
TOKEN = "ваш_токен_от_BotFather"
```

Токен можно получить у [@BotFather](https://t.me/BotFather) в Telegram.

### 5. Запуск бота

```bash
python main.py
```

Бот запустится в режиме long polling. Для остановки нажмите `Ctrl+C`.
